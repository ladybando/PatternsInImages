# PatternsInImages
Object detection using Google MLKit and Object Recognition

# Tools
- Kotlin 1.6.0
- Gradle 7.0.3
- Built with Android Studio Arctic Fox
- Kotlin Gradle Plugin 1.6.0


# SetUp
* `git clone` from Command Line
* Download the zip file
